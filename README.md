# GitMojo

## Install

```bash
git clone git@github.com:sixertoy/gitmojo.git
```

## Usage

- replace values in `./emojis.json`
- `npm run build`
- copy/move `./build/prepare-commit-msg` to your project's `.git/hooks` folder
