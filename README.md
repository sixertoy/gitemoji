# GitMojo

## Install

```bash
git clone git@github.com:sixertoy/gitmojo.git
```

## Usage

```bash
# replace values in `./emojis.json`
npm run start
# move `./build/prepare-commit-msg` to your project's `.git/hooks` folder
```
